# Analog Clock Trainer

**Analog Clock Trainer** is a small educational web app for practicing analog clock reading in English.

The app shows an analog clock and asks the learner to identify the correct time expression. It includes multiple difficulty levels, a kid-friendly multiple-choice mode, score tracking, feedback after each answer, and a final training summary.

The project was built as a focused learning tool and is suitable as a small portfolio project for UI, education, and interactive training workflows.

## Status

**Usable learning prototype / small educational side project**

The app is functional and can be used for clock-reading practice. It is not a certified educational product, but a small training app with a clear educational purpose.

## Live Demo

Try the app here:

[Open analog-clock-trainer](https://analog-clock-trainer.vercel.app)

## Features

* Analog clock display
* Multiple-choice training mode for kids
* Difficulty selection:

  * Kids Mode
  * Easy
  * Medium
  * Hard
* English time expressions such as:

  * `five o'clock`
  * `quarter to six`
  * `quarter past three`
* Score tracking
* Round-based training flow
* Immediate answer feedback
* Final completion screen
* Clean mobile-friendly UI
* Optional sound toggle

## Getting Started

### Prerequisites

* Node.js

### Install dependencies

```bash

```

### Run locally

```bash

```

### Build

